# TaskMate — Smart Study Planner

A full-featured Android study planner app built with Kotlin and Firebase.

## Features
- Firebase Authentication (Email/Password + Google Sign-In)
- Cloud Firestore real-time database
- Dashboard with tasks, subject progress, and upcoming exams
- Weekly timetable with color-coded schedule slots
- Subject tracker with chapter-level progress bars
- Progress tracker with circular goal rings
- Pomodoro focus timer with session tracking

## Tech Stack
- **Language:** Kotlin
- **Architecture:** MVVM + Repository Pattern
- **DI:** Hilt
- **Database:** Firebase Cloud Firestore
- **Auth:** Firebase Authentication
- **Async:** Kotlin Coroutines + Flow
- **UI:** Material Design 3, ViewBinding
- **Navigation:** Android Navigation Component

## Setup
1. Clone the repository
2. Add your own `google-services.json` from Firebase Console
3. Add your Google Web Client ID to `utils/Constants.kt`
4. Build and run
